# GUI Menu

## Описание

Это простая программа на Python с использованием библиотеки EasyGUI.

После запуска пользователю показывается меню с тремя кнопками:
- Кава
- Чай
- Выход

При выборе напитка открывается окно с названием, ценой и изображением.

Если выбрать "Выход", программа завершает работу.

## Используемые технологии

- Python
- EasyGUI

## Возможности

- Выбор напитка через кнопки.
- Отображение информации о напитке.
- Использование изображений (.gif).
- Завершение программы по нажатию кнопки "Выход".

## Как запустить

1. Установить библиотеку EasyGUI.
2. Поместить файлы изображений 2.gif и 4.gif рядом с программой.
3. Запустить файл Python.
